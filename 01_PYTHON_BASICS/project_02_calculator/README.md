# Simple Calculator (CLI)

This is a command-line calculator built using Python.
The user can select an operation from a menu and
perform calculations on two numbers.

## Features
- Addition
- Subtraction
- Multiplication
- Division
- Exit option
- Handles division by zero

## Concepts Used
- input() and output
- int() and float()
- if / elif / else conditions
- Arithmetic operators
- f-strings

## How to Run
1. Open terminal in this folder
2. Run the command:

   python main.py
