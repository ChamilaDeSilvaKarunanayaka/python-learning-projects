# Student Marks Manager (CLI)

This is a simple Python command-line program to manage student marks.
The user can add student marks, view all students, calculate the average,
and find the highest mark.

## Features
- Add student name and mark
- Display all students and their marks
- Calculate average mark
- Show highest mark
- Menu-driven program
- Runs continuously until exit

## Concepts Used
- Dictionary (key-value pairs)
- while loop
- if / elif / else conditions
- for loop
- input() and output
- int() and float()
- f-strings

## How to Run
1. Open terminal in this folder
2. Run the program using:

   python main.py
